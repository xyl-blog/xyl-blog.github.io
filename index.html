Moto X 刷机命令

发表于 2017-02-22   |   分类于 个人笔记   |     |   阅读次数 
进fastboot后，使用adb命令开始刷机：

//分区表
fastboot flash partition gpt.bin
fastboot flash motoboot motoboot.img
fastboot flash logo logo.bin
fastboot flash boot boot.img
fastboot flash recovery recovery.img
fastboot flash system system.img

//刷基带
fastboot flash modem NON-HLOS.bin
fastboot erase modemst1
fastboot erase modemst2
fastboot flash fsg fsg.mbn


fastboot erase cache
fastboot erase userdata
fastboot erase customize
fastboot erase clogo
//完成
fastboot oem fb_mode_clear
//重启
fastboot reboot

gpt.bin分区表品级若何确定?

　　用UltraEdit，翻开底包外的“gpt.bin”文件。按Ctrl+G，转到Ctrl+G搜刮框内输进：0x00006FFC。分区表品级便界说正在分区表的0x00006FFC的位置。
局限性：
　　解锁BL后，处于一级的分区表能够通刷，然而必需要找bootloader适配。没有适配否能招致间接变砖。国止解锁BL后，先要看看本人脚机的分区表品级，以就刷进其余版原的底包后能够刷归去。

注意：
    VICTARA_RETCN_XT1085_5.1_LPE23.32-70_cid11_CFC.xml  分级：5

比较详细的ffmpeg参数功能

ffmpeg项目由以下几部分组成:
ffmpeg 视频文件转换命令行工具,也支持经过实时电视卡抓取和编码成视频文件.
ffserver 基于HTTP(RTSP正在开发中)用于实时广播的多媒体服务器.也支持时间平移
fplay 用 SDL和FFmpeg库开发的一个简单的媒体播放器
libavcodec 一个包含了所有FFmpeg音视频编解码器的库.为了保证最优性能和高可复用性,大多数编解码器从头开发的.
libavformat 一个包含了所有的普通音视格式的解析器和产生器的库.
